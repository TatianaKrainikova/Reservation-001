Программа, считывает данные бронирования отеля
(номер комнаты, дату заселения и дату выезда)
и показывает данные бронирования, включая его
продолжительность в днях.

Затем вводятся новые даты заселения и выезда, обновляется
бронирование и снова выводится бронирование с обновленными данными.

Программа не должна принимать неверные данные для бронирования
в соответствии со следующими правилами:
- Изменения в бронировании возможны только на будущие даты.
- Дата выезда должна быть больше даты заселения.
